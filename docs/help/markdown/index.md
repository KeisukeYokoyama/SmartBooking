# Smart Booking 使い方ガイド

Smart Booking は、WordPressサイトに「予約フォーム」を追加できるプラグインです。
店舗・担当者・スケジュールを登録し、ショートコードを貼るだけで、来店予約・カウンセリング予約・面談予約などをサイト上で受け付けられます。

## 初めての方へ

最短ルートで予約受付を開始するには、次の順番で読み進めてください。

1. [インストール・有効化](installation.md)
2. [店舗の登録・管理](stores.md)
3. [担当者の登録・管理](staff.md)
4. [スケジュールの登録](schedule.md)
5. [予約フォームの設置](booking-form.md)

ここまで完了すれば、サイト上で予約を受け付けられる状態になります。

## ページ一覧

### 基本セットアップ

- [インストール・有効化](installation.md) — プラグインをWordPressに導入し、有効化する手順を解説します。
- [店舗の登録・管理](stores.md) — 予約を受け付ける店舗の情報を登録します。複数店舗にも対応しています。
- [担当者の登録・管理](staff.md) — 各店舗に所属する担当者（スタッフ）を登録します。
- [スケジュールの登録](schedule.md) — 予約可能な日付と時間枠を登録します。
- [スケジュールの一括登録](schedule-copy.md) — 1日分のスケジュールをコピーして、複数日に一気に反映します。
- [予約フォームの設置](booking-form.md) — ショートコードを使ってサイトにフォームを表示します。

### 運用・管理

- [予約の管理](reservations.md) — 受け付けた予約の一覧確認・ステータス変更・CSV出力を行います。
- [フォームのカスタマイズ](custom-fields.md) — 予約フォームに独自の入力項目を追加します。
- [デザインの設定](design.md) — ボタンや日付選択の色を、サイトのデザインに合わせて調整します。

### 通知・外部連携

- [メール通知の設定](email.md) — 予約受付時・確定時に送信されるメールの内容を設定します。
- [Googleカレンダー連携](google-calendar.md) — 予約をGoogleカレンダーに自動登録します。
- [ChatWork通知連携](chatwork.md) — ChatWorkルームへ予約通知を自動投稿します。
- [GTM（Googleタグマネージャー）連携](gtm.md) — 予約フローの各ステップでカスタムイベントを送信し、GA4 や Google広告でコンバージョン計測を行えます。
