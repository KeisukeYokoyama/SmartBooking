# Smart Booking 仕様修正: フロント予約フォームUIリデザイン

## 変更の背景

旧版 WP Smart Booking Lite のフロント予約フォームのUI/UXの方が新版より優れている。
旧版のデザインを踏襲しつつ、新版の機能（システムエンティティ、表示制御、カスタムフィールド等）を維持した形でUIを全面リデザインする。

参照資料: `docs/legacy-ui-handover/` 内の全ファイル

---

## 変更1: 画面構成の変更（最重要）

### 旧: ステップ分離式
```
[店舗選択] → [担当者選択] → [日付選択] → [時間選択] → [フォーム入力] → [確認画面] → [完了画面]
```

### 新: 1画面統合式
```
[店舗選択（別画面）] → [担当者選択（別画面）] → [メイン入力画面（1画面）] → [確認画面（別ページ）] → [完了画面（別ページ）]
```

### メイン入力画面の構成

**パターンA（日付→フォーム、デフォルト）:**
```
┌─────────────────────────────┐
│ [日付選択セクション]         │
│  - 日/月 トグル             │
│  - 日表示 or 月表示         │
│  - 時間スロット（縦並び）   │
├─────────────────────────────┤
│ [フォーム入力セクション]     │
│  - セクションタイトル       │
│  - 動的フィールド群         │
├─────────────────────────────┤
│ [予約内容の確認] ボタン      │
└─────────────────────────────┘
```

**パターンB（フォーム→日付）:**
```
┌─────────────────────────────┐
│ [フォーム入力セクション]     │
│  - セクションタイトル       │
│  - 動的フィールド群         │
├─────────────────────────────┤
│ [日付選択セクション]         │
│  - 日/月 トグル             │
│  - 日表示 or 月表示         │
│  - 時間スロット（縦並び）   │
├─────────────────────────────┤
│ [予約内容の確認] ボタン      │
└─────────────────────────────┘
```

### 店舗・担当者の扱い（変更なし）
- 店舗選択・担当者選択は別画面のまま維持
- スキップルール（1つならスキップ、show_*_front=OFF ならスキップ、is_system ならスキップ）も維持
- 店舗・担当者選択が完了したら、メイン入力画面に遷移

## 変更3: UIコンポーネント仕様

### 送信ボタン（プライマリ）
- 背景: var(--smb-front-color-button) デフォルト #f43f5e
- 文字: #fff / 16〜18px / 600
- 高さ: 60px / padding: 18px 30px / 角丸 8px
- ホバー: 背景 #e11d48 + translateY(-2px) + box-shadow 0 4px 12px rgba(244,63,94,0.3)
- 無効: opacity 0.7 / cursor not-allowed
- 送信中: テキスト非表示 + SVGスピナー表示

### 修正ボタン（アウトライン）
- 背景: 透明 / 枠: 2px solid #6c757d / 文字: #6c757d
- ホバー: 背景 #6c757d / 文字: #fff

### テキスト入力フィールド
- 幅: 100% / padding: 16px / 角丸: 8px / 枠: 1px solid #e1e8ed
- 背景: #fff / 16px
- フォーカス: 枠色をカスタムカラーに変更
- エラー: 枠 #e74c3c 2px + shadow 0 0 0 3px rgba(231,76,60,0.1) + scale 1→1.02→1（0.5s）

### ラジオ・チェックボックス
- カラム並び（gap 12px）
- 各オプション: padding 12px 20px / 枠 1px solid #e1e8ed / 角丸 8px
- ホバー: 枠 #3498db / 背景 #f8f9fa
- 選択時: 枠 #3498db / 背景 #ebf3fd

### 日付ピッカー（日表示）
- 横並びカード（80×80px / padding 15px 10px）
- 背景: #f3f4f6 / 角丸: 8px
- ホバー: #e5e7eb
- 選択: 背景 var(--smb-front-color-date-selected) デフォルト #374151 / 文字 #fff
- 土曜: #0ea5e9 / 日曜: #f43f5e / 無効: #9ca3af 取消線

### 日付ピッカー（月表示）
- 7列グリッド / gap 4px
- ヘッダー: 月タイトル 18px 600 + 左右ナビボタン（円形40px）
- セル: aspect-ratio 1
- 選択日: 背景 var(--smb-front-color-date-selected) / 文字 #fff
- 無効日: opacity 0.5 / #d1d5db / 取消線

### 日/月トグル
- コンテナ: 背景 #f3f4f6 / padding 4px / 角丸 6px
- ボタン: padding 8px 16px
- アクティブ: 背景 #fff + box-shadow（iOSセグメントコントロール風）

### 時間スロット
- 縦並びリスト（幅100% / padding 15px 20px / 中央寄せ）
- 枠: 1px solid #e1e8ed / 角丸 8px / 16px 500
- ホバー: 枠 #3498db / 背景 #f8f9fa
- 選択: 枠・背景とも var(--smb-front-color-time-selected) デフォルト #374151 / 文字 #fff
- 無効（締切/満席）: 背景 #f8f9fa / 文字 #6c757d / 接頭に「×」赤文字

### セクションヘッダー
- タイトル: 22px 600 #2c3e50 / margin-bottom 20px
- 説明: 16px #666 / margin-bottom 20px
- 必須バッジ: 背景 #ef4444 / 文字 #fff 10px / padding 2px 6px / 角丸 2px

### 確認画面のラベル＋値リスト
- カード: 背景 #fff / padding 20px / 枠 #e1e8ed / 角丸 8px
- 行: flex / padding 8px 0 / 区切り 1px solid #f1f3f4（最後行なし）
- ラベル列: 幅 120px / 13px / #6c757d
- 値列: 16px / #333 / 折返し可

### 完了画面
- 成功アイコン: ✓ 48〜60px #28a745
- エラーアイコン: ✗ 48〜60px #dc3545
- 予約詳細カード: 背景 #fff / 枠 #e1e8ed / 角丸 8px

## 変更しないもの

- 店舗選択・担当者選択の別画面表示
- スキップルール（is_system、show_*_front、1つならスキップ）
- 確認画面→完了画面の別ページ遷移

- 予約の競合防止ロジック
- ハニーポットスパム対策
- 管理画面のデザインタブでの色変更機能
- 既存のカスタムフィールド対応
